# Dashboard de Anúncios de Carros

## Descrição do Projeto

Este projeto é um aplicativo web desenvolvido com Streamlit que permite visualizar dados de anúncios de vendas de carros nos Estados Unidos.

## Funcionalidades

- **Histograma**: Visualize a distribuição da quilometragem (odometer) dos veículos
- **Gráfico de Dispersão**: Analise a relação entre quilometragem e preço dos veículos

## Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- Plotly Express

## Como Executar

1. Clone o repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute o aplicativo: `streamlit run app.py`
